# Spring의 리다이렉트

 - 컨트롤러에서 String 타입으로 리턴하면서 redirect:을 포함하면 리다이렉트를 수행한다
 - 리다이렉트 응답시 두 종류의 어트리뷰트를 추가할 수 있다
 
 
# RedirectAttribute

 - 리다이렉트시 클라이언트에서 GET방식 요청을 하게 되는데 해당 요청에 파라미터를 추가하는 방식
 - 주소값 뒤에 어트리뷰트가 추가되는 것을 눈으로 확인할 수 있다

# FlashAttribute

 - 리다이렉트시 데이터를 보내고 싶지만 주소에 노출시키고 싶지 않은경우에 사용하는 어트리뷰트
 - 실제로는 세션을 이용하여 구현되어 있다 (리다이렉트 응답시 세션에 데이터 저장 -> 리다이렉트 요청시 세션에 있는 데이터 활용 후 삭제)
 - @ModelAttribute(name)을 통해서 사용하거나 model.getAttribure(name)을 통해 사용할 수 있다
