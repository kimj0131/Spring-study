# AJAX(Async Javascript and XML)

 - 비동기 자바스크립트와 XML(+JSON)
 - 해당 웹 페이지를 새로고침하지 않고 요청을 보내 응답 받은 데이터를 페이지에 반영할 수 있는 비동기 요청
 - 주로 @RestController에 요청을 보내 데이터를 받아와서 활용하는 방식으로 사용한다
 - 서버로부터 데이터를 응답 받을때마다 이벤트가 발생하고 해당 이벤트를 발생하고 해당 이벤트를 활용하여 로직을 구성한다
 
 
# XMLHttpRequest

 - 자바스크립트의 비동기 요청 객체
 - 요청을 보내도 브라우저가 다른페이지로 넘어가지 않는다
 - 인스턴스를 생성한 후 해당 인스턴스에 여러가지 설정을 한 뒤 사용한다
 

# xhttp의 onreadystatechange 이벤트
 
 - xhttp 객체 생성 이후로부터 모든 상태가 변화할때 마다 발생하는 이벤트
 - readyState 1 : open()이 성공했을 때 (send()하기 전
 - readyState 2 : 요청에 대한 응답이 도착했을 때
 - readyState 3 : 응답에 대한 처리를 시작했을 때
 - readyState 4 : 응답에 대한 처리가 모두 완료되었을 때
 
 
# xhttp.open(method, URI)

 - 첫 번째 파라미터에는 요청 방식을 설정한다
   (GET, POST 뿐만 아니라 PUT, DELETE등의 메서드도 사용할 수 있다)
 - 두 번째 파라미터에는 요청 보낼 URI를 설정한다

# xhttp.send()

 - 설정을 모두 마친 후 서버로 비동기 요청을 보내는 메서드
 - GET방식을 제외한 나머지 요청에는 send()시에 데이터를 실어보낼 수 있다
 - 데이터는 문자열로만 전송할 수 있다.
 - 요청에 함께 실려가는 문자열이 어떤 내용인지 content-type을 통해 설명해야 한다
